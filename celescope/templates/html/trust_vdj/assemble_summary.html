Cells

Estimated number of cells: Number of cells which contain at least one chain (for TCR: TRA or TRB, for BCR: IGH, IGL or IGK).

Mean Read Pairs per Cell: Number of input reads divided by the estimated number of cells.

Mean Used Read Pairs per Cell: Mean number of reads used in assembly per cell-associated barcode.

Fraction Reads in Cells: Number of reads with cell-associated barcodes divided by the number of reads with valid barcodes.

Reads Mapped to Any V(D)J Gene: Fraction of reads that partially or wholly map to any germline V(D)J gene segment.

Reads Mapped to TRA: Fraction of reads that map partially or wholly to a germline TRA gene segment. For BCR, this should be one of [IGH, IGL, IGK].

Median TRA UMIs per cell: Median number of UMIs assigned to a TRA contig per cell. For BCR, this should be one of [IGH, IGL, IGK].

Reads Mapped to TRB: Fraction of reads that map partially or wholly to a germline TRB gene segment. For BCR, this should be one of [IGH, IGL, IGK].

Median TRB UMIs per cell: Median number of UMIs assigned to a TRB contig per cell. For BCR, this should be one of [IGH, IGL, IGK].
